Ceejay Portfolio React App

How to run:
1. Open this folder in VS Code.
2. Open the terminal.
3. Run: npm install
4. Run: npm run dev
5. Open the local link shown in the terminal.

Important:
- Put your resume PDF in this folder's public directory and name it resume.pdf
- Replace GitHub links in src/App.jsx with your project-specific repositories when ready
- You can add screenshots later by creating a public/images folder and referencing them in App.jsx
